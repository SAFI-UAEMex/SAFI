# Importante 

Para ser mas eficiente en Alium debes de saber usar `TODAS` las teclas rapidas, para toda accion o herramienta existe una tecla rapida, esto te vuelve un diseñador mas rapido y evitas entorpecerte buscando apartado por apartado lo que necesites

Pero aqui estarán escritas algunas de las mas importantes o mas usadas comunmente para todo dieseño, sobre todo enfocandonos en la parte del diseño de PCBs, no tanto de un esquematico

> [!NOTE]
> ### Teclas rapidas en archivos .PcbDoc

> P + G - POLYGON POUR

> T G H (en secuencia) - desaparecer visualmente POLYGON POUR

> T G E - regresarla

> T G A (una detrás de otra) - recalcular POLYGON POUR

> Ctrl+Shift+click izquierdo sobre una via o pad - resaltar todos los pads y pistas de una via

> L - view configuration

> Shift + S - Resaltar una sola capa

> "*" - para cambiar entre top y bottom
  
> T + U - Actualizar el cambio de un componente en el esquematico

> Ctrl + m - Crear una regla y revisar dimensiones entre centros de pads

> M + C + click izq - Atraer un componente

> 2 - Mientras ruteas sacar vias

> L - mientras ruteas despues de sacar una via para cambiar de capa
